OBJETIVOS
	O objetivo desse trabalho é desenvolver uma metodologia,..., para isso, utilizou-se tecnicas de aprendizagem profunda, redes neurais convolucionais, que são modelos biologicamente inspirados capazes de aprender caracteristicas de alto nível pela composição de baixo nível.


O objetivo deste trabalho é especificar um modelo de rede neural capaz de reconhecer
texto manuscrito em imagens digitais.
Os objetivos específicos do trabalho são:
a) reconhecer texto manuscrito em imagens digitais;
b) utilizar aprendizado profundo para reconhecimento de padrões visuais;
c) desenvolver um protótipo para validar o modelo apresentado.
